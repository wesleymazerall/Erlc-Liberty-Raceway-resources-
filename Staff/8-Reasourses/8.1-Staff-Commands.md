# 8.1 Staff Commands

## 8.1.1 Purpose

This document defines the approved use of staff commands and tools.

Staff commands exist to assist staff members in completing moderation and administrative responsibilities.

---

## 8.1.2 Command Usage

Staff members must only use commands for authorized purposes.

Commands must be used for:

- Moderation actions.
- Server management.
- Member assistance.
- Staff responsibilities.

---

## 8.1.3 Command Restrictions

Staff members must not use commands:

- For entertainment.
- To target members unfairly.
- To bypass server rules.
- Without a valid reason.

---

## 8.1.4 Command Responsibility

Staff members are responsible for actions performed through their account.

Incorrect or unauthorized command usage may result in review or disciplinary action.

---

## 8.1.5 Command Updates

Staff members are responsible for staying informed about changes to available commands and procedures.
